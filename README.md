# CampsiteCommander
This app is designed to help you with the arrangements of the items you may need for the camp.
The first screen is the splash screen which is basically the welcome screen.
After the welcome screen you find the Main screen where you will be entering your information,information being your item names,category,quantity and comments as well.
Thereafter you will be directed to the detailed view screen which shows all the items youve packed and how many items there are in total.
You will also find a "Back to Base" button in the detailed view screen which takes you back to the main screen in case theres something you may have missed.
